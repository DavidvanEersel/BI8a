Our project

This web based application was created for Radboud UMC to find new genes in between GenePanel updates.
GenePanel is a tool used in doctors to link genetic diseases to genes so patients can be treated better.

The primary function of this tool is to find new genes based on user input.
For example, if the user wishes to find articles related to deafness after the sixth of june 2021, the user can enter these settings in the text boxes and search for articles related to deafness.
These articles receive a score based on where the terms are positioned.
Scoring is used as a rough indicator as to how interesting the article is going to be to the researcher.
As in any case, each article gets the text mined hits.
Text mined hits include the genes, diseases and possible related mutations.
Found genes might have a relation to GenePanel and are shown in the results if a gene is found that there is a relation.

Django was used as a python framework for setting up the website.
This tool connects to the NCBI PubMed retrieving articles and uses the PubTator API as textmining algorithm.
