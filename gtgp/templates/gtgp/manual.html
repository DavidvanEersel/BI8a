Manual

Upload GenePanel

First you want to upload your GenePanel to the site. This can be done trough the uploadpage.
When you push the button "Browse" you're able to browse trough your files and select your GenePanel.
After that you the file will be displayed in the textarea above. You're able to make adjustments to file if necessary.
At last, when the file is ready to upload, press the button "Submit" and you're ready to go!

Gene Search

If you want to search, make sure you have uploaded a GenePanel file to the site.
When you have uploaded a file to the site you can enter some searchwords to the first textarea.
You'll have to use separators if you want to search for multiple words. This separator is ";".
If you to look up some synonyms you can use "(" and ")".
If you want to search for multiple words, you just place a separator(;) between the words.
For example, you want to search for deafness. But you're not sure if the use deafness or hearingloss and you also want to search on heart failure.
You can use this query: (deafness;hearingloss);heart failure.

If you want to search for specific genes, you can use the second textarea.
You can input the genes the same way as searching for keywords. For example you want the gene ZNF or zinc finger protein and polB gene.
You can use this query: (znf;zinc finger protein);polB.

If you want to search from a specific date, you can use the third box.
When you press it pops up a little calendar and you can select the date from where the website will search.

If you want to exclude a GenPanel from the result list, you are able to do so.
Fill in the last textarea with the GenPanel/GenPanels you want to exclude.
