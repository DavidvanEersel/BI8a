Manual

For uploading a newer version of GenePanel up to the website, one has to browse to the "Upload" page through the browsing menu.
When arriving at the "Upload" page, one must prepare a tab separated Tab Separated Value (.tsv)file containing the columns "Symbol_HGNC", "Aliases", "GenePanels_Symbol" and "GenePanel".
The user can give the data to the browser by either pasting the contents of the .tsv file in the "File" box or by clicking "Browse" and selecting the .tsv file.
When the user uploaded the data or pasted the data in the textbox, the user can resize the textbox by dragging on the bottom right part of the textbox, so the user can check the data.
After having uploaded the .tsv file or pasting it in the text box, the user clicks "Submit".
Now the web application knows the GenePanel data and is ready to use the GenePanel data.

Before making a search, is strongly advised to have an up-to-date GenePanel uploaded to the website, instructions in "Upload GenePanel manual".

To search PubMed, type the keywords in the 'keywords' textbox. This textbox can be resized by dragging on the bottom right part of the textbox. .
The query has to be written in a specific way. Every keyword has to be separated with the separator symbol, the semicolon (";").
If the user wishes to make 'and' statements, the user simply has to place a semicolon (;) behind each keyword. Spacebars are not required in between.
Would the user wish to make 'or' statements, the user has to place the or fields between brackets "(" and ")".
An example query would be: "(deaf;deafness;hearingloss);(ATP;ADP;AMP;cAMP;cyclicAMP);".
In this example, the web application will search for articles containing deaf, deafness, or hearling loss and ATP, ADP, AMP, cAMP or cyclicAMP.

Furthermore, the user has the option to search for a specific gene, which can be done by using the second textfield "Gene names".
For example, the user can enter the gene "P2X2" in the "Gene names" textbox.
Using this method by adding a gene in the "Gene names" textbox, the results will only contain articles that include the entered gene(s), in the example case, "F2X2".
More genes can be added by adding the semicolon(";") separator, like with the keywords.
The same way of using 'or' can be used by added using brackets "(" and ")" in the 'Gene names' textbox, which could be useful when searching for synonymous genes.
An example query for genes would be "(YARS;YARS2);UCN;".

Additionally, the user can search for articles after a specific publication date in the "Publication Date (After)" box.
This would be useful for searching for new articles after a new GenePanel update.
For example, the last GenePanel update was at 12-12-2021, the user can enter this date in the "Publication Date (After) box."
The searching tool will search only for articles published after 12-12-2021 and not before.

Finally, there is an option to exclude gene panel using the "Exclude gene panel" box.
To remove multiple known gene panel results, use the separator ";" semicolon.
One example of what the user can write in the "Exclude gene panel" is "HEMOS;OMIM;".
Another option is to exclude all known gene panels, which can be achieved by typing "ALL" in "Exclude gene panel".
Which can be useful when searching for completely unknown genes.

The result page is divided in multiple columns; the PMID's, Genes, Diseases, Mutations and Score.
PMID (PubMed ID) is a clickable link to the PubTator page where the hits have been found.
Genes are the genes found by the textmining algorithm in the article.
Diseases are the diseases found by the textmining algorithm in the PubMed article.
Mutations are the possible mutations the textmining algorithm has found in the PubMed article.
Score is the points the application has given to the hit.
This score is related to the number of hits found based on keywords.
For example, if the keyword is "deafness", the article will give points if the article contains the keyword "deafness".
If "deafness" is found in the title, the article will gain 5+ points.
If "deafness" is found in the abstract, the article will gain 2+ points.
If the gene P2X2 was in the keywords, it would give 1+ point.
